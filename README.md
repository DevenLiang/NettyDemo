# NettyDemo

 Netty是基于Java NIO client-server的网络应用框架，使用Netty可以快速开发网络应用
 更多用法请请跳转到https://github.com/netty/netty
 
 本项目基于Netty在Android平台所建项目，供大家参考，不足之处请提issue，大家一起解决。
 演示时，客户端在Const.java中请修改TCP服务端ip地址就行了，服务端通过切换通道，可以与多个客户端通信。
